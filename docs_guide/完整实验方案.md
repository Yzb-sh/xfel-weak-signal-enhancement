- [x] **1.2 编写Beamstop掩码生成模块**：根据真实实验mask的尺寸比例，生成标准化、可参数化（中心大小、支撑臂宽度）的二值掩码矩阵（0为有效，1为忽略）。

### 步骤2：搭建物理引导神经网络架构
- [ ] **2.1 实现部分卷积层**：编写 `PartialConv2d` 类，实现权重更新时自动根据输入掩码重新归一化的公式：$W' = W \cdot \frac{\sum M}{\sum (M \odot W)}$。
- [ ] **2.2 实现中心对称点卷积层**：编写 `CentroSymmetricConv` 类，将特征图沿中心点翻转后，与原特征图在通道维度拼接。
- [ ] **2.3 搭建核心U-Net结构**：构建包含4层下采样和4层上采样的标准U-Net，将普通的 `Conv2d` 替换为 `PartialConv2d`，在下采样路径中插入 `CentroSymmetricConv`。
- [ ] **2.4 定义数据预处理逻辑**：在模型的 `forward` 函数前，编写对输入noisy图的对数变换逻辑：$I_{log} = \log(I_{noisy} + \epsilon)$，并同步处理mask。
- [ ] **2.5 定义后处理逻辑**：在模型输出后，编写反变换逻辑：$I_{out} = \exp(I_{pred}) - \epsilon$，确保最终输出为线性尺度的去噪衍射图。

### 步骤3：损失函数设计与模型训练
- [ ] **3.1 实现基线损失函数**：编写仅计算有效区域（mask==0）平均像素误差的MSE Loss。
- [ ] **3.2 实现泊松负对数似然损失**：将模型反变换后的线性尺度输出与真值代入公式：$L_{PNLL} = \frac{1}{N}\sum (\hat{y}_i - y_i \log(\hat{y}_i + \epsilon))$，仅在有效区域计算。
- [ ] **3.3 实现频率加权损失**：对线性尺度的预测值和真值做2D FFT，按频率距离构建权重矩阵 $W(k) = |k|^2$，计算加权频域MSE。
- [ ] **3.4 实现自相关一致性损失**：对线性尺度的预测值和真值取模的平方做IFFT，计算实空间自相关函数在有效区域的L1距离。
- [ ] **3.5 实现复合物理引导损失**：将3.2至3.4加权组合（建议初始权重 $\lambda_{PNLL}=1.0, \lambda_{Freq}=0.1, \lambda_{Auto}=0.5$）。
- [ ] **3.6 编写双轨训练脚本**：创建两个独立配置，使用完全相同的优化器和学习率调度，分别使用3.1（基线模型）和3.5（物理模型）进行训练，保存各自的checkpoint。

### 步骤4：轻量级代理评估（用于日常调优）
- [ ] **4.1 实现去噪层基础指标**：计算测试集上去噪图与clean图的 PSNR 和 SSIM。
- [ ] **4.2 实现信噪比提升度指标**：在去噪前后的衍射图上，选取低频和高频两个环形区域，分别计算 $\frac{Mean_{signal}}{Std_{background}}$ 的提升倍数。
- [ ] **4.3 实现衍射层R-factor指标**：在倒易空间计算 $\frac{\sum ||\hat{F}| - |F||}{\sum |F|}$，仅在有效区域计算。
- [ ] **4.4 集成轻量级重构测试**：调用已有的重构算法，设定固定参数（如迭代200次），对每张测试图仅运行 **20次** 独立重构，记录平均R-factor和最终误差，不计算PRTF。

### 步骤5：无真值实验数据评估方案设计

### Prompt 2: 物理引导网络架构
```text
你现在是一名顶级的深度学习架构师。请在 `src/models/` 下实现参考论文的U-Net变体。代码必须严格遵循以下数学定义。

1. 创建 `layers.py`：
   - 实现 `PartialConv2d(in_ch, out_ch, kernel_size, stride, padding)`：
     继承 nn.Module。forward时接收输入 x 和 mask (0有效, 1无效)。
     卷积核计算：W * x，同时计算 W * mask。
     如果某像素位置的 mask_sum 为0，则该位置的输出和更新后的mask都设为0。
     如果 mask_sum > 0，则输出 = (W*x) / mask_sum。更新后的 mask = 1 (表示该位置已被有效信息覆盖)。
     
   - 实现 `CentroSymmetricConv(in_ch, out_ch)`：
     继承 nn.Module。forward时，将输入特征图沿中心点翻转（使用 torch.flip(x, dims=[-2, -1])），然后将原特征图与翻转后的特征图在 channel 维度拼接，最后通过一个 1x1 的 Conv2d (输入通道为 2*in_ch，输出为 out_ch)。

2. 创建 `unet.py`：
   - 实现 `PhysicsGuidedUNet(in_channels=1, out_channels=1)`：
     编码器：4层。每层包含 PartialConv2d(3x3, padding=1) -> BatchNorm -> GELU -> PartialConv2d(3x3) -> BatchNorm -> GELU -> CentroSymmetricConv -> MaxPool2d(2)。通道数变化：1 -> 64 -> 128 -> 256 -> 512。
     瓶颈层：PartialConv2d -> BatchNorm -> GELU -> PartialConv2d -> BatchNorm -> GELU。通道数 512 -> 1024。
     解码器：4层。每层包含 Upsample(scale_factor=2, mode='bilinear') -> 在channel维度 concat 跳跃连接 -> PartialConv2d -> BatchNorm -> GELU -> PartialConv2d -> BatchNorm -> GELU。通道数变化：1024 -> 512 -> 256 -> 128 -> 64。
     输出层：PartialConv2d(1x1) 输出单通道。
   - 在模型的 forward 函数中，严格加入预处理和后处理：
     输入 x (线性尺度), mask。
     x_log = torch.log(x + 1e-6)
     pred_log, updated_mask = self.unet(x_log, mask)
     pred_linear = torch.exp(pred_log) - 1e-6
     返回 pred_linear, updated_mask。
请确保 PartialConv2d 中的权重归一化计算使用 eps=1e-8 防止除零。不要使用现成的第三方库，纯手写这些层。
```

### Prompt 3: 复合损失函数与训练逻辑
```text
你现在是一名深度学习训练专家。请在 `src/losses/` 和 `scripts/` 下实现损失函数和训练循环。要求绝对精确的物理公式实现。

1. 创建 `src/losses/physics_loss.py`：
   - 实现一个类 `PhysicsGuidedLoss`，初始化参数 lambda_pnll=1.0, lambda_freq=0.1, lambda_auto=0.5, lambda_ssim=0.5。
   - forward(pred, target, mask)：
     注意：传入的 pred 和 target 必须是线性尺度（由模型反变换得出）。
     有效像素提取：仅保留 mask==0 的位置进行计算。
     
     (1) PNLL 损失：
     $L_{PNLL} = \frac{1}{N} \sum_{valid} (\hat{y}_i - y_i \log(\hat{y}_i + 1e-6))$
     
     (2) 频率加权损失 (Freq Loss)：
     对 pred 和 target 进行 2D FFT (使用 torch.fft.fftn，不需要shift)。
     构建与图像同尺寸的权重矩阵 W，其中每个像素的值等于其到频谱中心(0,0)的欧氏距离的平方。
     $L_{Freq} = \frac{\sum_{valid} W \cdot |\mathcal{F}(\hat{y}) - \mathcal{F}(y)|^2}{\sum_{valid} W}$
     
     (3) 自相关一致性损失：
     $A = \mathcal{F}^{-1}(|\mathcal{F}(\hat{y})|^2)$，真值同理。
     $L_{Auto} = \frac{1}{N} \sum_{valid} |A_{pred} - A_{target}|$
     
     (4) SSIM 损失：
     使用 pytorch_msssim 库（如果未安装请用 pip install pytorch-msssim 安装），计算 valid 区域的 1 - SSIM。如果无法切片计算，则对整图计算 1-SSIM。
     
     返回加权和。
```
