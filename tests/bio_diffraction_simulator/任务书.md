生物样品衍射数据仿真生成器 - 任务书

项目概述

核心目标：创建一个专用的数据生成流水线，为“基于深度学习的软X射线衍射去噪”项目生成用于模型训练的模拟数据。该流水线将产生大量模拟的、带有真实实验噪声的衍射图及其对应的“干净”衍射图和光束阻挡器掩膜。最终数据将用于训练一个类似文献《Denoising low-intensity diffraction signals using k-space deep learning》中的U-Net去噪网络。

首要原则：
1.  测试驱动开发：所有功能必须先使用极小的样本数（例如N=10）进行完整流程测试，确保每一步输出可视、逻辑正确、物理合理后，才能逐步扩大数据量。严禁一开始就生成大规模数据集。
2.  模块化：代码必须高度模块化，每个核心步骤（如物体生成、衍射模拟、噪声添加）应为独立、可配置的函数/类，便于单独调试和验证。
3.  物理一致性：所有模拟参数必须严格与上海光源软X射线自由电子激光实验的实际情况对齐，并在代码中显式验证。

第二部分：数据生成流程（必须严格按此顺序执行）

整个流程必须按以下步骤顺序编码，上一步的输出作为下一步的输入。每个步骤必须先在小样本（N=10）上测试通过。

步骤1: 构建参数化生物形态模型（BioSampleGenerator类）

目标：生成代表大肠杆菌的二维实空间投影密度图 obj。
• 基础形状：“胶囊体”（两端为半圆的矩形）。

• 尺寸：长度 L ∈ [1.0e-6, 3.0e-6] 米，直径 D ∈ [0.5e-6, 1.0e-6] 米。

• 形态变异：必须实现表面Perlin噪声、内部高斯斑点（3-10个）、圆形低密度液泡（1-3个），以及正常/分裂/弯曲三种细胞状态。

• 输出：一个2D numpy数组，尺寸为 (train_size, train_size)，数据类型 float32，值域 [0, 1]，代表相对投影密度。

步骤2: 数据多样性增强（DataAugmentor类）

目标：对步骤1生成的 obj 进行随机变换，扩充数据多样性。
必须实现的变换（按顺序应用）：
1.  随机旋转：均匀随机旋转 [0, 360) 度，使用双线性插值。
2.  随机平移：在 [-0.1*train_size, 0.1*train_size] 像素范围内随机平移。
3.  随机缩放：在 [0.9, 1.1] 范围内均匀随机缩放。

步骤3: 模拟衍射图生成（DiffractionSimulator类）

目标：计算物体投影 obj 的“干净”衍射强度 I_clean。
物理公式与实现：
# obj 是实空间投影密度
A = np.fft.fft2(obj)                    # 复振幅
A = np.fft.fftshift(A)                  # 将零频移到中心
I_clean = np.abs(A)**2                  # 衍射强度

物理尺度校准（关键！）：
• 计算实空间像素尺寸：dx_real = detector_target_pixel_size

• 计算倒易空间像素尺寸：dq = 1 / (train_size * dx_real)

• 计算最大空间频率：q_max = dq * (train_size // 2)

• 输出：I_clean，尺寸 (train_size, train_size)，float32 类型。

【新增】步骤4: 衍射强度归一化（IntensityNormalizer类）

目标：将 I_clean 归一化，使其总和为1，为添加物理正确的泊松噪声做准备。
def normalize_intensity(I_clean):
    total_intensity = np.sum(I_clean)
    I_clean_normalized = I_clean / total_intensity  # 总和 ≈ 1
    return I_clean_normalized


步骤5: 应用随机掩膜（RandomMaskApplier类）

目标：模拟探测器的不规则缺陷。
• 以50%的概率决定是否应用随机掩膜。

• 如果应用，生成随机形状的二进制掩膜（圆形、矩形、不规则形状的组合）。

• 将掩膜区域在 I_clean_normalized 中的值设为0。

• 记录掩膜位置，用于后续处理。

步骤6: 应用实验噪声模型与光束阻挡器（NoiseAndBeamstopApplier类）

目标：将 I_clean_normalized 转化为与真实数据高度相似的 I_noisy。
1.  泊松噪声（核心，必须物理正确）：
    def add_poisson_noise(I_clean_normalized):
        I_sc = np.random.uniform(EXP_CONFIG['poisson_I_sc_range'][0], EXP_CONFIG['poisson_I_sc_range'][1])
        expected_photons = I_clean_normalized * I_sc  # 期望光子数分布
        I_noisy = np.random.poisson(expected_photons).astype(np.float32) # 施加泊松噪声
        return I_noisy, I_sc
    
2.  高斯读出噪声：I_noisy = I_noisy + np.random.normal(0, EXP_CONFIG['gaussian_read_noise_sigma'], I_noisy.shape)
3.  坏点与坏线：按概率随机添加。
4.  光束阻挡器：
    ◦ 从指定文件加载二进制掩膜 beamstop_mask。

    ◦ 创建随机宽度的渐变过渡区。

    ◦ 应用阻挡：完全阻挡区置零，渐变区按权重衰减。

• 输出：I_noisy (噪声衍射图)，beamstop_mask (掩膜图)。

步骤7: 最终预处理、组装与保存

1.  对数缩放（必需）：I_clean_log = np.log10(1 + I_clean); I_noisy_log = np.log10(1 + I_noisy)
2.  标准化（必需）：使用训练集统计量对 I_noisy_log 进行标准化（均值=0，标准差=1），并同步应用于 I_clean_log。
3.  组装与保存为HDF5文件：

    my_bio_diffraction_dataset.h5
    ├── train/
    │   ├── input      (shape: [N_train, 585, 585], dtype: float32)  # I_noisy_norm
    │   ├── target     (shape: [N_train, 585, 585], dtype: float32)  # I_clean_norm
    │   └── mask       (shape: [N_train, 585, 585], dtype: float32)  # beamstop_mask
    ├── val/           (结构同train)
    ├── test/          (结构同train)
    └── config/        (存储所有配置参数和元数据)

3.2 小样本测试流程（必须遵循）

1.  运行测试：python test_small_sample.py --num_samples 10 --output_dir ./test_output
2.  可视化验证：生成并检查每一步的中间结果图（物体、干净/噪声衍射图对比、掩膜）。
3.  物理验证：脚本必须打印并验证所有计算出的物理量（dx_real, dq, q_max等），确保自洽。
4.  统计验证（关键）：
    ◦ 泊松噪声：在非光束阻挡区域验证 方差(I_noisy) / 均值(I_noisy) ≈ 1。

    ◦ 数据分布：验证标准化后的 I_noisy_norm 数据近似符合标准正态分布（均值≈0，标准差≈1）。

3.3 关键验证点

1.  尺寸一致性：所有数组必须是 (585, 585)。
2.  物理单位：确保所有计算使用国际单位制（米）。
3.  泊松噪声统计：验证 方差/均值 ≈ 1。
4.  光束阻挡器：必须与实验形状完全一致，渐变过渡平滑。
5.  数据范围：标准化后数据应近似符合标准正态分布。
