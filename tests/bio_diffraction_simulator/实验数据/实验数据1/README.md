# 实验数据噪声分析总结

## 一、分析所用的数据

### 1. 实验数据
- **文件名**: `Ecoli_AB_20min_7897_background_subtraction_result.mat`
- **变量名**: `data_sb`
- **尺寸**: 4096 × 4096
- **说明**: 背景扣除后的衍射数据，用于分析噪声水平

### 2. Mask数据
- **文件名**: `Ecoli_AB_20min_7897_20260322_140309_missing-latest.mat`
- **变量名**: `mask`
- **尺寸**: 4096 × 4096
- **说明**: beamstop中心光束阻挡区域的mask，值为1表示阻挡区域，值为0表示有效数据区域

### 3. 分析代码
- **文件名**: `noise_analysis.py`
- **说明**: 噪声分析脚本，用于确定泊松噪声模拟参数 I_sc

---

## 二、Mask在噪声分析中的作用

### 关键作用
1. **排除beamstop区域**: beamstop区域的值会干扰噪声统计
2. **定义有效数据区域**: 只在mask=0的区域进行噪声统计
3. **选择背景区域**: 在选择背景区域时排除mask覆盖的部分

### 代码示例
```python
# 加载数据
data = sio.loadmat(data_file)['data_sb']
mask = sio.loadmat(mask_file)['mask']

# 只取有效区域的数据
valid_mask = (mask == 0)
I_valid = I[valid_mask]

# 选择背景区域时排除mask
bg_corner = I[100:400, 100:400]
bg_corner_valid = bg_corner[mask[100:400, 100:400] == 0]
```

## 四、最终建议

### 推荐 I_sc 范围

| 场景 | I_sc 范围 | 说明 |
|------|-----------|------|
| 匹配实验数据噪声 | ~1e9 | 背景σ ≈ 10.9 |
| **论文目标场景** | **1e4 ~ 1e7** | 低强度、高噪声 |
| 建议实验范围 | [1e4, 1e5, 1e6, 1e7, 1e8] | 覆盖高噪声到中等噪声 |

### 使用方法

```bash
# 运行噪声分析脚本
python noise_analysis.py
```
